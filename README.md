# 📞 Personal Phone Call – Real-Time Video, Audio & Chat Application

## 📌 Project Overview

Personal Phone Call is a real-time communication web application that allows users to communicate securely through video calls, audio calls, and instant messaging. The application is designed for personal use and provides a simple, fast, and user-friendly interface for connecting with friends.

Each user must first create an account and log in. After successful authentication, the system automatically generates a **unique user code**. This unique code acts as the user's identity and can be shared with others to add friends. Once two users are connected, they can exchange messages, make audio calls, and initiate high-quality video calls in real time.

## 🚀 How It Works

1. Register a new account and log in.
2. The system automatically generates a unique user code.
3. Share your unique code with friends.
4. Add friends by entering their unique code.
5. Start private conversations through:

   * 💬 Instant Messaging
   * 🎤 Audio Calls
   * 📹 Video Calls
6. Enjoy secure, real-time communication with a smooth and responsive experience.
